# Urbano-Tienda
Prototipo del Sprint 1 para el proyecto “Urbano Tienda”, desarrollado en el ramo Ingeniería de Requisitos. Incluye el catálogo dinámico, visualización de productos, selección de tallas y colores, y módulo de carrito. Proyecto realizado por el equipo: Darynka Sau Garcés, Nitsi Sepúlveda y Sofía González.
